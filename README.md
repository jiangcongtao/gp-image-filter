# 图片滤镜器 🎨

一个功能强大、界面美观的单页面应用（SPA），用于为图片应用创意滤镜效果。支持拖拽上传、实时预览、批量处理和多种滤镜效果。

## ✨ 功能特性

### 🖼️ 图片处理
- **拖拽上传**：支持拖拽图片到指定区域，或点击浏览选择文件
- **批量处理**：一次上传多张图片，批量应用滤镜效果
- **格式支持**：支持 JPG、PNG、WebP 格式
- **图片压缩**：自动压缩处理后的图片，优化文件大小

### 🎨 滤镜效果
提供 6 种精心调校的滤镜效果：
- **鲜艳**：增强色彩饱和度，让图片更加生动
- **复古**：营造怀旧氛围，添加暖色调
- **黑白**：经典黑白效果，支持强度调节
- **清新**：明亮清新的色调，适合风景照
- **晨光**：模拟清晨阳光的温暖色调
- **傍晚**：营造黄昏时分的浪漫氛围

### ⚙️ 参数调节
每种滤镜都支持精细的参数调节：
- **强度**：控制滤镜效果的强弱程度
- **暖度**：调整图片的冷暖色调
- **饱和度**：控制颜色的鲜艳程度

### 👁️ 预览功能
- **实时预览**：调整参数时实时查看效果
- **对比预览**：拖拽分隔线对比原图与滤镜后的效果
- **全屏预览**：点击图片查看大图预览

### 🌐 多语言支持
- 中文（简体）
- English

### 🎯 用户体验
- **深色模式**：支持亮色/深色主题切换
- **响应式设计**：适配桌面和移动设备
- **本地存储**：自动保存用户偏好设置（语言、主题、滤镜参数）
- **进度提示**：批量处理时显示处理进度
- **Toast 通知**：操作反馈提示

### 💾 下载功能
- **单张下载**：预览时下载单张处理后的图片
- **批量下载**：一键下载所有处理后的图片（ZIP 压缩包）

## 🚀 快速开始

### 使用方法

1. **打开应用**
   - 直接在浏览器中打开 `index.html` 文件
   - 或使用本地服务器运行（推荐）

2. **上传图片**
   - 拖拽图片到上传区域
   - 或点击上传区域选择文件

3. **选择滤镜**
   - 从 6 种滤镜中选择一种
   - 调整参数滑块自定义效果

4. **应用滤镜**
   - 点击"应用滤镜"按钮处理所有图片
   - 或点击单张图片的预览按钮实时查看效果

5. **下载图片**
   - 在预览界面点击下载按钮保存单张图片
   - 或在图片列表中点击"下载所有"批量下载

### 使用本地服务器（推荐）

```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Node.js (需要安装 http-server)
npx http-server -p 8000

# 使用 PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 🎨 滤镜算法说明

### 鲜艳滤镜 (Vibrant)
- 增强色彩饱和度
- 提高对比度
- 优化色彩平衡

### 复古滤镜 (Vintage)
- 添加暖色调（增加红色和黄色）
- 降低亮度
- 增强对比度

### 黑白滤镜 (B&W)
- 灰度转换
- 支持强度调节（0-100%）
- 保持细节层次

### 清新滤镜 (Fresh)
- 提高亮度
- 增强蓝色和绿色通道
- 降低暖色调

### 晨光滤镜 (Morning)
- 模拟清晨阳光的暖色调
- 提高亮度
- 增强黄色和橙色

### 傍晚滤镜 (Evening)
- 营造黄昏氛围
- 增强红色和橙色
- 降低整体亮度

## ⚙️ 配置说明

应用会自动保存以下用户偏好到浏览器本地存储：
- 语言设置（中文/英文）
- 主题模式（亮色/深色）
- 当前选择的滤镜
- 滤镜参数（强度、暖度、饱和度）

清除浏览器缓存或使用隐私模式将重置这些设置。

## 📝 使用技巧

1. **批量处理**：上传多张图片后，调整好滤镜参数，点击"应用滤镜"一次性处理所有图片
2. **实时预览**：在预览界面调整参数滑块，可以实时看到效果变化
3. **对比查看**：在预览界面拖拽中间的分隔线，可以对比原图和处理后的效果
4. **参数记忆**：应用会记住你上次使用的滤镜和参数，下次打开时自动恢复

## 🔧 开发说明

这是一个单文件应用，所有代码都在 `index.html` 中：
- CSS 样式在 `<style>` 标签中
- JavaScript 逻辑在 `<script>` 标签中
- 使用原生 JavaScript，无外部依赖（JSZip 通过 CDN 动态加载）

### 自定义滤镜

要添加新的滤镜效果，需要：
1. 在 `filterDefaults` 对象中添加默认参数
2. 在 HTML 中添加滤镜选项按钮
3. 实现滤镜算法函数（如 `applyNewFilter`）
4. 在 `applyFilterToImage` 函数中添加调用逻辑
5. 在翻译对象中添加多语言标签
